# Class-Libraries

#如何开发一个属于自己的类库
－基于webpack进行资源构建
－使用es6语法进行开发
－将包输出为umd规范的文件
－使用mocha和chai进行测试
－使用eslint来进行代码检查
－基于npm scripts来进行集成开发

#基本开发环境
-全局环境：node & npm
-命令行：ternimal & shell
-版本控制：git
-编辑器：atom
-浏览器：chrome
-注册npm账号
－注册github账号

#创建项目的基本结构
－src开发源码
－lib编译后的资源
－test测试用例
－.gitignore
-LICENSE
-README.md
-package.json
-webpack.config.js
-karma.conf.js
-.babelrc 使用babel的时候配置的一些环境变量
-.eslint.js
